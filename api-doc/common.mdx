ISPKEY 开放接口公共参数

**请求基础路径**
https://open.ispkey.com

**请求参数**

| 名称 | 类型 | 必选 | 说明 |
|---|---|---|---|
| appKey | string | 是 | 用户的AppKey |

```url
https://open.ispkey.com/接口路径?appKey=xxx
```

**备注**
除资源提取接口，其他所有请求都需要传`appKey`。
