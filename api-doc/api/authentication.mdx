`GET` /api/get_proxy_ip

**[提取配置页面，点击跳转](https://ispkey.com/admin/resources/fetch/fetchApiWin?auth=api)**

**请求参数**

|名称|类型|必选|说明|
|---|---|---|---|
|User|string| 是 |认证账号|
|pass|string| 是 |认证密码|
|region|string| 否 |国家编码|
|area|string| 否 |地区ID|
|time|string| 否 |保持时间|
|count|string| 是 |提取数量|
|protocol|string| 否 |协议|
|format|string| 是 |格式（json、txt）|
|split|string| 否 |分隔符（txt专用）|
|splice|string| 否 |拼接格式（txt专用）|
|proxyType|string| 否 |代理服务器类型,1或2|

注：代理服务器类型：1-非亚洲【us.ispkey.net】2-亚洲【sg.ispkey.net】

```url
https://open.ispkey.com/api/get_proxy_ip?user=xxx&pass=xxx&count=10&protocol=socks5&format=json&proxyType=1
```

**响应结果**

|名称|类型|必选|说明|
|---|---|---|---|
|» code|integer|true|响应代码|
|» msg|string|true|响应消息|
|» data|[array]|true|响应数据|
|»» ip|string|true|代理地址|
|»» pass|string|true|代理端口|
|»» port|string|true|认证密码|
|»» user|string|true|认证账号|
