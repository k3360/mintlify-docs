`GET` /api/region/getList

**请求参数**

| 名称 | 类型 | 必选 | 说明 |
|---|---|---|---|
| pid | int | 与region二选一 | 国家ID |
| region | string | 与pid二选一 | 国家编码 |

```url
https://open.ispkey.com/api/region/getList?pid=75
https://open.ispkey.com/api/region/getList?region=US
```


**响应结果**

| 名称 | 类型 | 说明 |
|---|---|---|
| code | int | 响应代码 |
| msg | string | 响应消息 |
| data | list  | 地区列表 |
