---
title: "添加白名单"
---

**post**    /white/addWhitelist

**请求体**

| 名称 | 类型 | 必选 | 说明 |
|---|---|---|---|
| appKey | string | 是 | 密钥 |
| ipAddr | string | 是 | IP地址 |

<Tabs>
<Tab title="cURL">
```bash
curl --request POST \
--url http://www.ispkey.com/api/white/addWhitelist \
--header 'Content-Type: application/json' \
--data '{
"appKey": "<string>",
"ipAddr": "<string>"
}'
```
</Tab>
<Tab title="Python">
```Python
import requests

url = "http://www.ispkey.com/api/white/addWhitelist"

payload = {
    "appKey": "<string>",
    "ipAddress": "<string>"
}
headers = {"Content-Type": "application/json"}

response = requests.request("POST", url, json=payload, headers=headers)

print(response.text)
```

import (
