`GET` /whitelist/add

**请求参数**

| 名称 | 类型 | 必选 | 说明 |
|---|---|---|---|
| appKey | string | 是 | 密钥 |
| ip | string | 是 | IP地址，格式：IPv4 |

```url
https://open.ispkey.com/api/whitelist/add?appKey=xxx&ip=12.12.12.12
```

**响应结果**

| 名称 | 类型 | 说明 |
|---|---|---|
| code | int | 响应代码 |
| msg | string | 响应消息 |

<Tabs>

```JSON
}
```
</Tab>
