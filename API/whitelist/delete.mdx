`POST` /white/removeWhitelist

**请求体**

| 名称 | 类型 | 必选 | 说明 |
|---|---|---|---|
| appKey | string | 是 | 密钥 |
| ipAddr | string | 是 | IP地址 |

```JSON
{
    "appKey": "<string>",
    "ipAddr": "<string>"
}
```


**响应体**

| 名称 | 类型 | 说明 |
|---|---|---|
| code | int | 响应代码 |
| msg | string | 响应消息 |
| data | any  | 响应数据 |
